# FixFlow Frontend (Angular)

Ce projet est la partie frontend de FixFlow, développée avec **Angular v21** et **Tailwind CSS**.

## Configuration

Le projet communique avec le backend via `http://localhost:8080/api/v1`.

## Commandes

- `npm install` : Installer les dépendances
- `npm start` : Lancer le serveur de développement (`http://localhost:4200`)
- `npm run build` : Compiler pour la production

## Design System

Les couleurs et animations personnalisées sont définies dans `tailwind.config.js`.
Les composants utilisent principalement le "Glassmorphism" (fonds semi-transparents et flous de backdrop).
